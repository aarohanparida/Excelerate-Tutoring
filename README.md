# Excelerate-Tutoring

This repository contains the static website for Excelerate Tutoring.

Pages:

* **Home** (`index.html`)
* **Courses** (`courses.html`)
* **Meet The Tutors** (`tutors.html`) — overview of our volunteer tutors
* **Contact** (`contact.html`)
* **Register** (`register.html`)

Each page uses shared styling (`home.css`) and small JavaScript helpers for
sticky navigation, parallax backgrounds, and interactive elements.
